## Local development

Want to run this project locally?

### Installation

1. Clone the repository:

```bash
git clone https://github.com/lindajensen/portfolio-v2.git
cd portfolio-v2
```

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```
